Base IA

Mes outils IA, No Code et LLMs

Notion

Productivité Débutant

Workspace tout-en-un pour notes, documentation, bases de données et organisation.

GitHub

Développement Débutant

GitHub est une plateforme qui héberge du code source et permet de versionner ses fichiers, collaborer avec d'autres développeurs et automatiser des tâches via GitHub Actions.

n8n

Automatisation Intermédiaire

Plateforme d’automatisation open source permettant de connecter des applications et créer des workflows sans coder.

Airtable

Base de données Débutant

Airtable est un outil no-code qui ressemble à un tableur, mais qui fonctionne comme une vraie base de données visuelle. Il permet de structurer des informations, créer des vues, suivre des processus et collaborer sans coder.

Apify

Scraping Intermédiaire

Apify est une plateforme de scraping et d’automatisation qui permet de récupérer automatiquement des données sur Internet via des robots appelés Actors.

Netlify

Hébergement Débutant

Netlify est une plateforme d’hébergement et de déploiement de sites web modernes et d’applications front-end.

Make

Automatisation Débutant

Plateforme no-code d’automatisation visuelle permettant de connecter des applications et créer des scénarios automatisés.

Clay

Automatisation Intermédiaire

Outil de prospection et d'enrichissement de données B2B qui connecte plus de 100 sources (LinkedIn, Apollo, Clearbit, Hunter...) pour construire des listes de leads qualifiées et automatiser l'outreach personnalisé grâce à l'IA.

Lovable

No-Code Intermédiaire

Générateur d'applications web full-stack par IA -- tu décris ce que tu veux en langage naturel et l'outil génère le code frontend et backend complet, déployable en quelques minutes.

Slack

Productivité Débutant

Plateforme de messagerie professionnelle organisée en canaux thématiques. Permet la communication d'équipe en temps réel, l'intégration d'outils externes et la réception de notifications automatisées depuis des workflows.

Claude

LLM Débutant

LLM d'Anthropic, connu pour son raisonnement rigoureux, son respect des instructions et sa capacité à traiter de très longs documents. Disponible en trois niveaux : Haiku (rapide), Sonnet (équilibré), Opus (puissant).

ChatGPT

LLM Débutant

LLM d'OpenAI, le plus utilisé au monde. GPT-4o est le modèle phare avec vision, voix et accès web intégré. Ecosystème très riche : Custom GPTs, plugins, Canvas, DALL-E pour les images.

Gemini

LLM Débutant

LLM de Google, nativement intégré à Google Workspace (Docs, Sheets, Gmail, Drive). Gemini 1.5 Pro dispose d'une fenêtre de contexte de 1 million de tokens. Très fort en multimodal (texte, images, audio, vidéo).

Perplexity

LLM Débutant

Moteur de recherche IA qui combine LLM et recherche web en temps réel. Répond aux questions avec des sources vérifiables et citées. Excellent pour la veille, la recherche factuelle et remplacer Google sur les questions complexes.

Claude Code

LLM Intermédiaire

Outil de développement IA d'Anthropic intégré directement dans le terminal. Basé sur Claude, il comprend le contexte complet d'un projet, lit et modifie des fichiers, exécute des commandes et permet de coder, déboguer et déployer sans quitter la ligne de commande.

GitHub Copilot

LLM Intermédiaire

Assistant IA de coding intégré directement dans l'éditeur de code -- il suggère des complétions, génère des fonctions entières, répond à des questions sur le code et peut réaliser des tâches de développement de façon semi-autonome.
