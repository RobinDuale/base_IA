● Presentation

A quoi sert ChatGPT ?

LLM d'OpenAI, le plus utilisé au monde. GPT-4o est le modèle phare avec vision, voix et accès web intégré. Ecosystème très riche : Custom GPTs, plugins, Canvas, DALL-E pour les images.

● Usage

Quand utiliser ChatGPT

Pour un usage généraliste quotidien, surtout quand on a besoin de recherche web en temps réel, d'analyse d'images, de génération d'images, ou qu'on veut utiliser un Custom GPT spécialisé.

● Economique

Ce que comprend la version gratuite

Accès à GPT-4o mini (illimité) et GPT-4o (limité). Navigation web incluse. Génération d'images DALL-E limitée. Pas d'accès aux Custom GPTs avancés ni à la mémoire complète.

● Points forts

Pourquoi utiliser ChatGPT

Interface la plus connue et la plus intuitive. Accès web natif (Bing). Custom GPTs et GPT Store. Vision (analyse d'images). Voice mode avancé. Mémoire persistante entre les conversations. Enorme communauté.

● Limites

Les limites a connaitre

Hallucinations plus fréquentes que Claude sur les tâches précises. Comportement moins prévisible sur des instructions complexes. API plus coûteuse à grande échelle. Données potentiellement utilisées pour l'entraînement.

● Usage

Avec quels outils ChatGPT est-il complementaire ?

n8n / Make (automatisation via API OpenAI), DALL-E (génération d'images intégrée), Zapier (intégrations no-code).

● Economique

Modele economique de ChatGPT

Freemium. ChatGPT Plus à 20$/mois : GPT-4o illimité, DALL-E illimité, Custom GPTs, mémoire avancée. API OpenAI : GPT-4o mini ($0.15/M tokens), GPT-4o ($2.50/M).

● Economique

Quand passer a la version payante

Dès qu'on utilise ChatGPT quotidiennement et qu'on atteint les limites GPT-4o, ou qu'on a besoin des Custom GPTs et de la génération d'images sans restriction.

● Alternatives

Quelles alternatives a ChatGPT ?

Claude (Anthropic), Gemini (Google), Mistral, Llama (Meta)

Scenarios d'usage

Debutant

Générer des variantes de textes marketing ou débloquer une rédaction difficile -- idéal pour les premières ébauches de contenu, les posts LinkedIn ou les descriptions produit.

Intermediaire

Clay enrichit une liste de prospects avec leurs spécificités sectorielles -> ChatGPT via API génère des messages de prospection personnalisés par segment -> n8n intègre ces messages dans une séquence email automatisée.

Avance

ChatGPT analyse le sentiment de retours clients (emails, données Airtable) -> calcule un score de satisfaction prédictif -> n8n déclenche des alertes Slack si la tendance se dégrade -> Apify scrape les mentions publiques de la marque pour compléter l'analyse -> rapport consolidé dans Notion.
