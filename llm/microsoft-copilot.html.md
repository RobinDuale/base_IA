# Microsoft Copilot

> Assistant IA généraliste de Microsoft, propulsé par GPT-4o -- accessible gratuitement sur le web, dans Windows 11 et Edge, et en version avancée intégrée à Microsoft 365 (Word, Excel, PowerPoint, Outlook, Teams).

## Résumé

- Type : LLM
- Catégorie : LLMs
- Niveau : Débutant
- Gratuité : Accès illimité sur copilot.microsoft.com sans compte Microsoft. Avec compte gratuit : historique des conversations et personnalisation. Génération d'images DALL-E 3 (limitée). Disponible dans Windows 11 et Microsoft Edge. Pas de limite de messages sur la version web.
- Site officiel : https://copilot.microsoft.com
- Tags : IA, Workflow
- Fiche HTML : https://ia.duale.fr/llm/microsoft-copilot.html


## Quand utiliser Microsoft Copilot

Quand tu travailles dans l'écosystème Microsoft et veux de l'IA sans changer d'environnement. Ou pour une recherche web rapide avec sources dans Edge -- bonne alternative gratuite à Perplexity pour la veille.


## Pourquoi utiliser Microsoft Copilot

Gratuit et sans compte requis pour un usage basique. Intégration native dans l'écosystème Microsoft (Windows, Office, Edge). Recherche web en temps réel intégrée avec sources citées. Génération d'images via DALL-E 3 incluse. La version M365 est très puissante pour automatiser les tâches dans Word, Excel, Outlook et Teams.


## Limites de Microsoft Copilot

Moins performant que Claude ou ChatGPT sur des tâches complexes de raisonnement. La version M365 est chère (30$/mois par utilisateur, en plus de l'abonnement Office existant). Qualité variable selon les intégrations. Moins flexible que les APIs concurrentes pour les développeurs.



## Exemples et workflows

Réunion Teams -> Copilot génère le compte-rendu automatiquement -> actions envoyées dans Outlook. Ou : données dans Excel -> Copilot analyse et génère des graphiques -> export en PowerPoint en un clic.


## Modèle économique

Freemium avec deux niveaux payants : - Gratuit : accès illimité sur copilot.microsoft.com, intégré dans Windows et Edge - Copilot Pro : 20$/mois -- accès prioritaire à GPT-4o, génération d'images illimitée, Copilot dans les apps Office - Microsoft 365 Copilot : 30$/mois par utilisateur -- intégration complète dans Word, Excel, PowerPoint, Outlook, Teams (nécessite un abonnement M365 existant)


## Quand payer

Rester sur le gratuit pour la recherche web et les tâches générales. Copilot Pro (20$/mois) si tu utilises intensivement la génération d'images ou veux l'accès prioritaire. M365 Copilot (30$/mois) uniquement si tu travailles quotidiennement dans les apps Office et veux automatiser les réunions Teams, la rédaction Word et l'analyse Excel.


## Alternatives

ChatGPT, Claude, Gemini, Perplexity


## Complémentaire avec

Microsoft 365 (Word, Excel, PowerPoint, Outlook, Teams -- intégration native), Edge (assistant de navigation intégré), n8n (peut interagir avec l'API Microsoft Graph pour automatiser des tâches Office), Notion (alternative pour ceux hors écosystème Microsoft)


## Scénario débutant

Poser une question à Microsoft Copilot dans Edge ou Windows -- il répond avec des sources web récentes, comme Perplexity, mais intégré directement dans le navigateur ou l'OS sans ouvrir un autre onglet.


## Scénario intermédiaire

Dans Outlook, Copilot rédige un résumé d'un long fil d'emails -> propose une réponse adaptée au ton de la conversation -> dans Teams, il génère un compte-rendu de réunion avec les points d'action identifiés.


## Scénario avancé

Copilot M365 analyse un fichier Excel de données commerciales -> génère des graphiques et identifie des tendances -> rédige un rapport Word avec les insights -> prépare les slides PowerPoint -> envoie un résumé par Outlook. Tout sans quitter l'écosystème Microsoft.
