# Perplexity

> Moteur de recherche IA qui combine LLM et recherche web en temps réel. Répond aux questions avec des sources vérifiables et citées. Excellent pour la veille, la recherche factuelle et remplacer Google sur les questions complexes.

## Résumé

- Type : Outil IA et No-Code
- Catégorie : LLMs
- Niveau : Débutant
- Gratuité : Recherches illimitées avec le modèle standard. Accès web en temps réel. 5 recherches Pro par jour (modèles avancés). Upload de fichiers non disponible en gratuit.
- Site officiel : https://www.perplexity.ai
- Tags : Search, Monitoring, IA
- Fiche HTML : https://ia.duale.fr/outils/perplexity.html


## Quand utiliser Perplexity

Quand on cherche une réponse factuelle et sourcée sur un sujet récent ou technique. Remplace avantageusement Google pour les questions complexes qui nécessitent une synthèse, pas juste une liste de liens.


## Pourquoi utiliser Perplexity

Réponses sourcées et vérifiables -- chaque affirmation est liée à une source. Recherche web en temps réel. Interface épurée et rapide. Spaces pour organiser les recherches par thème. Pas d'hallucination sans source (ou presque).


## Limites de Perplexity

Pas un LLM généraliste : limité à la recherche et au Q&A. Moins performant que Claude ou ChatGPT pour la rédaction longue, le code ou les tâches créatives. Hallucinations possibles malgré les citations de sources.




## Modèle économique

Freemium. Perplexity Pro à 20$/mois : modèles avancés au choix (GPT-4o, Claude, Gemini), recherches illimitées, upload de fichiers, Spaces collaboratifs.


## Quand payer

Si on fait de la veille régulière et qu'on veut accéder aux meilleurs modèles avec upload de fichiers. Le gratuit couvre largement un usage occasionnel.


## Alternatives

ChatGPT avec web (OpenAI), Gemini (Google), You.com, Phind (focus dev)


## Complémentaire avec

Notion (archiver les recherches), Airtable (structurer les données trouvées), n8n (automatiser des requêtes Perplexity via API).


## Scénario débutant

Rechercher des informations récentes sur un marché ou un concurrent avec sources citées et vérifiées -- zéro hallucination, réponses actualisées en temps réel contrairement aux LLMs à date de coupure fixe.


## Scénario intermédiaire

Configurer une veille quotidienne sur des thèmes définis (concurrents, régulation IA, nouveaux outils) -> Perplexity génère un résumé avec sources -> n8n l'envoie chaque matin dans Slack et le stocke dans Notion.


## Scénario avancé

Perplexity identifie des entreprises en croissance (levées, recrutements, actualités) -> Claude qualifie les signaux et les classe par pertinence -> Clay enrichit les décideurs identifiés -> Slack notifie l'équipe avec les leads priorisés et leur contexte -> Airtable trace les actions commerciales.
