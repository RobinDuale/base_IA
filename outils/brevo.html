● Presentation

A quoi sert Brevo ?

Brevo (ex-Sendinblue) est une plateforme d'emailing transactionnel et marketing. Elle permet d'envoyer des emails automatiques (confirmations, notifications, alertes) via API REST ou SMTP, avec 300 emails gratuits par jour sans carte bancaire.

● Presentation

Son role dans l'ecosysteme IA

Brevo est la couche de communication email dans un écosystème No-Code / automatisation. Il se connecte à n8n pour envoyer des emails déclenchés par des événements : soumission de formulaire, validation de compte, publication de contenu.

● Usage

Quand utiliser Brevo

Utilise Brevo quand tu dois envoyer des emails transactionnels automatisés (confirmation d'inscription, réinitialisation de mot de passe, notification de workflow) sans dépasser 300 emails/jour. C'est le bon choix pour les projets solo ou les petites apps où tu veux un SMTP fiable et gratuit, intégrable facilement dans n8n ou via API. Si ton volume augmente ou que tu veux retirer le logo Brevo, passe à un plan payant.

● Economique

Ce que comprend la version gratuite

300 emails transactionnels par jour, contacts illimités, accès API, SMTP, templates, analytics de base. Aucune carte bancaire requise.

● Points forts

Pourquoi utiliser Brevo

  • 300 emails/jour gratuits sans limite de contacts
  • API REST simple et bien documentée
  • Node natif dans n8n (aucune configuration complexe)
  • SMTP disponible pour intégration dans tout langage
  • Templates d'emails avec éditeur visuel
  • Analytics : taux d'ouverture, clics, bounces
  • Domaine d'envoi personnalisé possible
  • Pas de carte bancaire pour le plan gratuit
● Limites

Les limites a connaitre

  • Logo Brevo visible dans les emails du plan gratuit
  • 300 emails/jour max en gratuit
  • Fonctionnalités avancées réservées aux plans payants
  • Délivrabilité parfois inférieure aux solutions premium
● Usage

Avec quels outils Brevo est-il complementaire ?

n8n (déclencheur et orchestrateur des emails), Notion (source de données), GitHub Actions (pipeline de déploiement)

● Economique

Modele economique de Brevo

Freemium. Plan gratuit : 300 emails/jour, contacts illimités, logo Brevo. Plans payants à partir de 9$/mois pour supprimer le logo et augmenter les volumes.

● Economique

Quand passer a la version payante

Quand le volume dépasse 300 emails/jour ou pour supprimer le logo Brevo des emails.

● Alternatives

Quelles alternatives a Brevo ?

Resend (100 emails/jour gratuits, API moderne), Mailgun (5000 emails/mois gratuits 3 mois), SendGrid (100 emails/jour gratuits), Postmark (payant, excellente délivrabilité)
