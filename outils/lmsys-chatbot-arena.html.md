# LMSYS Chatbot Arena

> Une plateforme de benchmarking open-source pilotée par la communauté pour évaluer et classer les grands modèles de langage (LLM) via des tests en aveugle (A/B testing) et des évaluations humaines.

## Résumé

- Type : Outil IA et No-Code
- Catégorie : LLMs
- Niveau : Débutant
- Gratuité : Gratuit
- Site officiel : https://chat.lmsys.org/
- Tags : IA, Research, Chat
- Fiche HTML : https://ia.duale.fr/outils/lmsys-chatbot-arena.html


## Quand utiliser LMSYS Chatbot Arena

Dès que vous hésitez sur le choix d'un modèle pour un projet IA ou que vous souhaitez tester les dernières sorties sans payer d'abonnement.


## Pourquoi utiliser LMSYS Chatbot Arena

Classement Elo fiable basé sur des préférences humaines réelles - Accès gratuit aux modèles les plus performants du marché (GPT-4o, Claude 3.5 Sonnet, Gemini 1.5 Pro) - Transparence sur les performances relatives des modèles - Environnement neutre pour comparer les capacités de raisonnement


## Limites de LMSYS Chatbot Arena

Pas de persistance des données (pas d'historique de chat longue durée) - Interface axée sur le test et non sur la productivité quotidienne - Soumis à la disponibilité des serveurs et aux files d'attente lors des pics de trafic



## Exemples et workflows

Tester un prompt complexe sur 3 modèles différents pour choisir le plus pertinent avant de l'intégrer dans une API.

## Scénario débutant

Vous posez une question complexe à deux modèles anonymes et votez pour la meilleure réponse afin de découvrir quels modèles sont les plus performants.


## Scénario intermédiaire

Vous utilisez le mode 'Direct Chat' pour comparer les réponses d'un modèle spécifique (ex: GPT-4o vs Claude 3.5) sur une tâche de rédaction technique.


## Scénario avancé

Utiliser les données du leaderboard pour justifier techniquement auprès d'une direction le choix d'un modèle spécifique dans une stratégie d'implémentation d'IA en entreprise.
