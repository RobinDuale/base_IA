Description

Workspace tout-en-un pour notes, documentation, bases de données et organisation.

Avantages

Très flexible, excellent pour centraliser l’information.

Limites

Moins puissant qu’Airtable pour les workflows complexes.

Modele économique

Freemium avec fonctionnalités avancées payantes.

Quand payer ?

Quand la collaboration, l’IA ou les limites de stockage deviennent importantes.

Alternatives

Coda, Obsidian, Confluence

Notes personnelles

Servira de système central de connaissance. Déjà utilisé comme source de données pour le projet Base IA : l'API Notion alimente un script Node.js qui génère les pages HTML du site ia.duale.fr. Chaque modification dans Notion déclenche un rebuild automatique via n8n et GitHub Actions.
