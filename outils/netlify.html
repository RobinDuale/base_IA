Netlify

Hébergement Débutant
Site officiel →

Description

Netlify est une plateforme d’hébergement et de déploiement de sites web modernes et d’applications front-end.

Avantages

Très simple pour déployer rapidement un site moderne, bon pour les débutants, très utilisé avec les outils front-end modernes.

Limites

Ce n’est pas un outil métier ni une base de données. Netlify sert surtout à publier et héberger une interface web.

Modele économique

Freemium SaaS avec plans payants selon le nombre de membres, le trafic, les fonctionnalités avancées et les besoins professionnels.

Quand payer ?

Rester sur le gratuit pour apprendre et publier des prototypes. Passer au payant quand le site devient professionnel, critique, collaboratif ou reçoit davantage de trafic.

Alternatives

Vercel, Cloudflare Pages

Notes personnelles

Netlify n’est pas comparable à Airtable : Airtable organise le business et les données, Netlify publie une interface ou un site sur Internet. Les deux peuvent être complémentaires.
